This folder contains solutions to the Bike Classification Problem

The Following items are present in the folder:

1. The directory contains 3 main folders:
        data-We access the data for processing from this folder
        Mountain_Bike-This is the original folder containg 50 images of Mountain Bikes
        Road_Bike-This is the original folder containg 50 images of Road Bikes

2. Bike_Classifier.ipyb - is an iPython Notebook, which contains the executed python code
3. Bike_Classifier.py - Contains the python code (This was executed on Ubuntu 16.04 using Keras with Tensorflow Backend)
4. model_plot.png - Tensorboard Output of the Recurrent Convolutional Neural Network Structure
